Write-Once Memory codes

Write-Once Memory codes or WOM codes have been introduced by Rivest&Shamir in 1982.
These pages bundle the results of my quest in this area. The following is provided:

  • The report "A heuristic approach to find short efficient WOM codes"
  • A list of short 2-write binary WOM codes
  • A list of short multiple-write binary WOM codes
  • The sample program "WOM_finder" as used to find 2-write binary WOM codes with arbitrary decoding map
  • Two sample programs demonstrating the practical use of "best effort codes" to obtain high effective coding rates in classical WOM and cyclical storage applications
